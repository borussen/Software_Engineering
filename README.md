## 🛠 개발 환경

### Frontend

| 카테고리            | 기술명            |
| ------------------- | ----------------- |
| 언어                | JavaScript (16)   |
| 프레임워크          | React (18.2.0)    |
| 컴포넌트 라이브러리 | Chakra-UI (2.3.6) |

### Backend

| 카테고리   | 기술명                                     |
| ---------- | ------------------------------------------ |
| 언어       | Python (3.9)                               |
| 프레임워크 | Django (4.1), Django REST Framework (3.14) |
| DB         | SQLite (3)                                 |
